# Personal Portfolio Website

This is my personal portfolio website built using **React** and **Tailwind CSS**.  
It showcases my skills, projects, and contact information in a clean and responsive layout.

The purpose of this project is to practice modern frontend development while creating a professional online presence.

## Features

- Responsive design (mobile, tablet, desktop)
- Modern UI using Tailwind CSS
- Component-based architecture with React
- Sections for:
  - Hero / Introduction
  - About Me
  - Skills
  - Projects
  - Contact

## What I Learned

This project helped me understand and practice:

- React fundamentals (components, props, JSX)
- Tailwind CSS utility-first styling
- Responsive UI design
- Project structure in React applications
- Writing clean and reusable components
